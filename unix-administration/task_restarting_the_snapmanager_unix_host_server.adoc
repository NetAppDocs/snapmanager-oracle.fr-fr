= Redémarrage du serveur hôte UNIX SnapManager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vous pouvez redémarrer le serveur sur un hôte UNIX à l'aide de la CLI.

. Pour redémarrer le serveur, entrez la commande suivante :
`smo_server restart`
